# Código de Conduta do Projeto

Este projeto adota o Pacto de Código de Conduta do Contributor Covenant.

- Versão: 2.1 — https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- Tradução PT-BR: https://www.contributor-covenant.org/pt-br/version/2/1/code_of_conduct/

## Nossa promessa

Criar um ambiente aberto e acolhedor, sem assédio, onde todos se sintam seguros para contribuir.

## Nossos padrões

Exemplos de comportamento que contribuem para um ambiente positivo:

- Uso de linguagem acolhedora e inclusiva
- Respeito por diferentes pontos de vista e experiências
- Aceitação construtiva de críticas

Comportamentos inaceitáveis incluem:

- Linguagem ou imagens sexualizadas ou assédio
- Ataques pessoais ou políticos
- Publicar informações privadas de outras pessoas

## Aplicação

Relatos de abuso, assédio, ou comportamento inaceitável podem ser enviados via Issues (marcadas como "security"/"abuse")
ou por contato direto com os maintainers quando disponível. As denúncias serão revisadas e investigadas com a devida
confidencialidade na medida do possível.

## Atribuição

Este Código de Conduta é adaptado do Contributor Covenant, versão 2.1.
